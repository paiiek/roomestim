## 3. 구현

### Phase 1 — Core 신규
- `roomestim/reconstruct/predictor.py` NEW (~280 LoC):
  - `is_rectilinear_shoebox(room) -> bool`
  - `_shoebox_dimensions_m(room) -> (L, W, H)`
  - `_shoebox_surface_areas_and_alphas(room) -> (6-tuple, 6-tuple)`
  - `_shoebox_per_band_alphas(room) -> (6-tuple, dict[band, 6-tuple])`
  - `predict_rt60_default(room, area_dict, *, prefer_ism, max_order) -> RT60Prediction`
  - `predict_rt60_default_per_band(...)` — per-band variant.
  - `RT60Prediction` dataclass (frozen).
  - `_polygon_area_3d`, `_room_volume`, `_shoelace_2d` 코어에 duplicate
    (web 의존성 차단; layering).
- `roomestim/reconstruct/__init__.py` — re-export 5 public symbols.

### Phase 2 — Web report 통합
- `roomestim_web/report.py` `AcousticReport` 에 4 신규 필드:
  `default_rt60_500hz_s`, `default_rt60_per_band_s`, `default_predictor_name`,
  `default_predictor_rationale`.
- `to_json_dict()` 4 필드 직렬화.
- `compute_acoustic_report()`: `predict_rt60_default*` 호출, 실패 시 Eyring
  fallback degradation (rationale에 실패 사유 기록).

  ISM 분기 시 per-band green bar 추가.

### Phase 3 — 테스트

  - predict_rt60_default_per_band: 6 밴드 + 500 Hz 일치.
  - prefer_ism=False → Eyring.
  - ISM/Eyring 런타임 invariant (ADR 0009 ism ≥ eyring - 1e-6).

  - PredictorName Literal 집합 = {'image_source', 'eyring'}.
