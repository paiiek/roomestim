## 수정 항목 (확정 결함)

### FIX-1 (MAJOR-1, ✅실측) — ISM 기본 predictor 저흡음 미수렴 → Eyring 하한 불변식 위반
- 증거: α=0.05 shoebox 5×4×2.8 에서 Eyring=1.944, ISM@max_order=50=1.675(위반), @80=2.195, @200=2.327. `predictor.py:428,523` default 50.
- **설계 결정**: 저수준 `image_source_rt60(max_order=N)` 는 **그대로**(deterministic, `test_image_source.py` 가 order 고정으로 검증). **고수준 `predict_rt60_default`/`_per_band` 만 적응적**으로:
  1. max_order 를 [50, 100, 200] 으로 단계 상향하며 ISM 계산.
  2. ISM 결과가 `eyring_rt60 - 1e-6` 이상이 되는(불변식 충족) 첫 order 채택. (수렴 = EDC 가 threshold 도달.)
  3. cap(200)에서도 여전히 < eyring 이면 **eyring 으로 fallback** + rationale 문자열에 "ISM non-converged at max_order=200 → Eyring fallback" 명시(honesty).
  4. predictor rationale 문자열에 실제 사용 order 반영 (`per-band ISM (max_order=...)`).
- 테스트: α=0.05 에서 `predict_rt60_default` 가 불변식(result ≥ eyring-1e-6) 충족하는 회귀 테스트 추가.
- 주의: ADR 0030(default predictor) §Status-update 로 이 변경 기록. 성능: 200³ lattice 는 저흡음 방에서만 트리거(고흡음은 50에서 수렴 → 비용 불변).

### FIX-2 (MAJOR-2, ✅코드확인) — 비-shoebox binaural DOA az/el 축 스왑
- 증거: `binaural.py:243-253` _to_pra: shoebox=`[x,height,depth]`, extrusion=`[x,depth,height]`. DOA(`:338-342`) `az=atan2(rel[0],rel[2])`,`el=atan2(rel[1],horiz)` 는 shoebox 규약 가정 → extrusion 에서 rel[1]=depth, rel[2]=height 로 뒤바뀜. listener_pos=pra frame(`:261`).
- **수정**: DOA 계산을 경로별 올바른 축으로:
  - shoebox: up=rel[1], front=rel[2], side=rel[0]
  - extrusion: up=rel[2], front=rel[1], side=rel[0]
  - `az=atan2(side, front)`, `el=atan2(up, sqrt(side²+front²))`.
- 테스트: 비-shoebox(L-shape 또는 extrusion) 방에서 알려진 위쪽 image 가 el>0, 측면 image 가 |az|≈90 이 되는지 — 렌더러 실제 경로 통과(공식 재구현 금지). 기존 `test_binaural_doa_axis_mapping` 의 가짜신뢰 보강.
