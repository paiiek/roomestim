## TASLP-MISC — DONE at v0.6.0

### What landed

Six of seven ACE rooms gained one synthesised `Surface(material=MISC_SOFT,
kind="floor")` whose Newell-area is integrand-preserving at 500 Hz:

```
area_misc_soft = Σ_pieces count_i * A_500_i / a_misc_soft_500
```

with `a_misc_soft_500 = MaterialAbsorption[MISC_SOFT] = 0.40` (v0.5.0 row).

Per-room synthesised areas (m²):

| Room | Furniture | Synthesised MISC_SOFT area |
| --- | --- | ---: |
| Office_1 | 4 office_chair | 5.00 |
| Office_2 | 6 office_chair, 1 bookcase | 8.25 |
| Meeting_1 | 14 office_chair | 17.50 |
| Meeting_2 | 30 office_chair, 6 table | 39.00 |
| Lecture_1 | 60 lecture_seat, 20 table | 72.50 |
| Lecture_2 | 100 lecture_seat, 35 table | 121.25 (strip-clip; Lecture_2 floor 13.6 × 9.29 m) |
| Building_Lobby | (excluded) | n/a |

Per-piece α₅₀₀ (m² Sabines per piece): office_chair 0.50, stacking_chair
0.15, lecture_seat 0.45, table 0.10, bookcase 0.30 — see ADR 0013 for the
table-row citations (Vorländer 2020 §11 / Appx A primary; Beranek 2004 Ch.3
Table 3.1 cross-check for `lecture_seat`).

### Empirical effect — v0.5 → v0.6 perf doc deltas (500 Hz Sabine)

| Room | v0.5 Sabine (s) | v0.6 Sabine (s) | Δ Sabine (s) | v0.5 err (s) | v0.6 err (s) |
| --- | ---: | ---: | ---: | ---: | ---: |
| Building_Lobby | 2.022 | 2.022 | 0.000 | +1.425 | +1.425 |
| Lecture_1 | 1.762 | 0.686 | −1.076 | +1.201 | +0.125 |
| Lecture_2 | 0.673 | 0.435 | −0.238 | −0.670 | −0.908 |
| Meeting_1 | 0.499 | 0.410 | −0.089 | +0.072 | −0.017 |
| Meeting_2 | 0.468 | 0.395 | −0.073 | +0.061 | −0.012 |
| Office_1 | 0.864 | 0.704 | −0.160 | +0.486 | +0.327 |
| Office_2 | 0.862 | 0.631 | −0.231 | +0.179 | +0.179 |

Highlights:

- **Lecture_1**: Sabine 500 Hz error collapses from +1.201 s to +0.125 s —
  the v0.4 Finding 4 hypothesis ("bare-walls model under-counts absorption
  budget") is empirically supported. The 60 lecture seats + 20 tables
  contribute 72.5 m² of MISC_SOFT-class surface.
- **Lecture_2**: Sabine 500 Hz under-prediction deepens (−0.670 →
  −0.908 s). This is consistent with the F3 hypothesis (the actual ceiling
  is more reflective than `ceiling_acoustic_tile`) — adding furniture
  absorption shifts predicted RT60 lower, which increases under-prediction
  on a room whose measured RT60 is already > 1.3 s. F3 stays DEFERRED;
  v0.6 does not address ceiling material indeterminacy.
- **Office_1 / Office_2 / Meeting_1 / Meeting_2**: modest drops (0.07 –
  0.23 s) bringing 500 Hz errors from over-prediction of 0.06 – 0.49 s
  toward 0 (Office_2 still over-predicts by +0.179 s; Meetings now
  marginally under-predict by ~0.01 s).
- **Building_Lobby**: unchanged (`Building_Lobby` not in
  `_FURNITURE_BY_ROOM`; helper returns None; surface count remains 6).

### Eyring monotonicity

per band continues to hold in `tests/test_e2e_ace_challenge_rt60.py` with

## What stays DEFERRED to v0.7+

- F1 walls / ceiling materials (INDETERMINATE; not TASLP-blocked; no
  canonical source).
- F3 — Lecture_2 ceiling material hypothesis (canonical path closed;
  needs non-canonical evidence).
- F4a — `MaterialAbsorptionBands` coefficient revision (D14 5b
  pre-condition unchanged).
- **`lecture_seat` α₅₀₀ revision (NEW v0.7+ work item; reverse-trigger
  fired empirically by Lecture_2 over-correction at v0.6)**:
  retrospective code-review flagged Lecture_2 as crossing both common
  reverse-trigger heuristics (|err|/measured = 67.6% > 50%; |err| =
  0.908 s > 0.50 s absolute). The locked α₅₀₀ = 0.45 m² Sabines per
  empty lecture seat is plausibly the occupied-row from the cited
  textbook (Beranek 2004 Ch.3 Tab 3.1 distinguishes occupied vs
  unoccupied; Vorländer 2020 §11 has a similar split). Lecture_1 is
  correctly predicted at v0.6, so the α is not universally wrong —
  the issue is interaction with Lecture_2's ceiling material (F3
  hypothesis). v0.7 ADR options: (a) split lecture_seat into
  `lecture_seat_occupied` / `lecture_seat_empty` piece types; (b)
  revise locked α₅₀₀ to ~0.20–0.25 with new textbook citation; (c)
  hold lecture_seat unchanged and confirm Lecture_2 ceiling = drywall
  (F3) via non-canonical evidence — see F3 above. Strict ADR 0013
  §Reverse-if uses "textbook re-read shows > 30% drift" language
  not "field validation > 30% drift", so this fires empirically
  (informational) rather than letter-of-the-law.
