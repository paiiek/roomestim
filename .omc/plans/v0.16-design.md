## 3. 구현 Phase (high-level)

### Phase 1 — `evolve_room` + `evolve_surface` helpers
- `roomestim/model.py`에 `evolve_room(room, *, surfaces=...) -> RoomModel` 추가
- `evolve_surface(surf, *, material=...) -> Surface` — MaterialLabel 입력 시 absorption_500hz/bands 자동 lookup
- 기존 frozen 유지; 모든 mutation은 새 instance

### Phase 2 — Web UI 재질 override
- `roomestim_web/app.py` 신규 Surface Override Tab
- Surface list (kind + index + current material)
- Per-row Material dropdown (10 MaterialLabel options)
- "Recompute Acoustics" 버튼 → AcousticReport 재생성

### Phase 3 — 2D 블루프린트 export
- `roomestim_web/blueprint.py` NEW
- matplotlib top-down view (floor polygon + wall lines + listener area + speakers)
- PNG (raster) + SVG (vector) 두 포맷
- "Blueprint" 탭 추가 OR Setup PDF에 페이지 추가

### Phase 4 — Engine 검증 토글
- CLI: `roomestim export --validate-engine [PATH]` / `--no-engine-validation`
- Web UI: 사이드바 "Standalone YAML (no engine schema check)" 체크박스
- ENV var `SPATIAL_ENGINE_REPO_DIR` 호환 유지 (precedence: CLI > env > default)
