# roomestim v0.8 Release — Design Plan

- **Date**: 2026-05-09 (drafted) / 2026-05-09 (scope locked)
- **Author**: oh-my-claudecode:planner (auto mode; orchestrator brief
  pre-locked Scope A bracketing + Scope B per-band MAE snapshot)
- **Predecessor**: v0.7.0 (`b56ea7c` — WFS CLI ergonomics + ADR 0014
  Building_Lobby coupled-space exclusion; D18)
- **Status**: SCOPE LOCKED — Scope A (Lecture_2 ceiling bracketing +
  `lecture_seat` α occupied/unoccupied split) + Scope B (per-band ex-BL
  MAE snapshot test). v0.8 is a SUBSTANTIVE numerical-experiment
  release answering the v0.7 critic verdict "v0.7 was borderline ADR
  theatre; v0.8 must ship a residual-shrinking experiment".
- **Mode**: SHORT (sensitivity bracketing harness + golden snapshot +
  ADR 0015 + bookkeeping; no architecture redesign, no schema flip,
  no enum extension, no library-level coefficient revision, no
  predictor change, no `_FURNITURE_BY_ROOM` mutation, no
  `MaterialAbsorption{,Bands}` mutation).

This plan operationalises the v0.7 audit's "What stays DEFERRED to
v0.8+" set restricted to the leverage-ranked candidates from the v0.7
critic review:

- **Candidate #1 — Lecture_2 bracketing experiment** (ceiling material
  swap + `lecture_seat` α occupied/unoccupied split).
- **Candidate #3 — Per-band ex-BL MAE snapshot test** (frozen golden
  file; future PRs editing predictor behaviour must update + justify).

What is **NOT** in scope (deferred per existing decisions / ranked-
lower by critic; explicit invalidation rationale below):

- **Candidate #2 — F4a constrained sensitivity sweep at 2 / 4 kHz**
  (`MaterialAbsorptionBands[wall_painted]` + `[ceiling_drywall]`) is
  **DEFERRED to v0.9+**. Gating: v0.8 Scope A first answers whether a
  single-coefficient ceiling/seat swap closes Lecture_2; if yes,
  Candidate #2 is unnecessary; if no, v0.9 enters the broader
  per-band sensitivity sweep with a stronger prior over which rows
  matter.
- F1 walls / ceiling materials — INDETERMINATE per ADR 0012 (canonical
  evidence path closed).
- F3 baseline-changing ratification — v0.8 ships bracketing as
  *sensitivity analysis only*; no v0.6/v0.7 numerical-baseline edit.
- Hard-floor subtype confirmation — needs non-canonical evidence.
- Coupled-space predictor (Cremer / Müller) — ADR 0014 alt-considered.
- Stage-2 schema flip — D8 / OQ-4 unchanged; A10 lab capture pending.
- Millington-Sette per-surface predictor — ADR 0009 alt-considered.
- 8 kHz octave band — ADR 0008 reverse criterion not met.
- PyPI / submodule — D11 unchanged.
- New `MaterialLabel` enum entry — none warranted by Scope A or B.

The local tag `v0.8.0` is local-only (D11 precedent: only v0.1.1 is on
remote).

### §2.1 Scope A — `_build_room_model` overrides hook

**File**: `roomestim/adapters/ace_challenge.py`.

Add a small private dataclass + an optional `overrides=` kwarg to
`_build_room_model`. The library default-path stays byte-equal: when
`overrides is None`, all existing tests see identical surfaces.

```python
@dataclass(frozen=True)
class _RoomBuildOverrides:
    """Per-variant overrides used by Scope-A bracketing test (v0.8).

    None of these mutate library defaults — the overrides are
    constructed at the test layer and passed in for one synthesis
    call. The default _build_room_model(...) path is byte-equal to
    v0.7 when overrides is None.
    """
    ceiling_label: MaterialLabel | None = None       # V1 / V3 / V4
    seat_alpha_500: float | None = None              # V2 / V3
    seat_alpha_bands: tuple[float, ...] | None = None  # V2 / V3 (len 6)
```

Wire into `_build_room_model`:

1. Add `overrides: _RoomBuildOverrides | None = None` keyword arg.
2. If `overrides and overrides.ceiling_label is not None`, replace
   `ceiling_mat` with `overrides.ceiling_label` BEFORE the ceiling
   `Surface(...)` is constructed. This affects only the ceiling
   surface; floor + walls remain `_MATERIAL_MAP`-derived.
3. Per-variant lecture_seat α — does NOT touch
   `_PIECE_EQUIVALENT_ABSORPTION_*` dicts. Instead, when
   `overrides and overrides.seat_alpha_500 is not None` and
   `room_id` is in `_FURNITURE_BY_ROOM`, the helper for the synthetic
   MISC_SOFT surface uses a *local* α: the test passes both the 500
   Hz scalar and the 6-band tuple; `_misc_soft_surface_from_furniture`
   gains a sibling helper `_misc_soft_surface_from_furniture_with_alpha(...)`
   that computes area against the supplied alpha (not
   `MaterialAbsorption[MaterialLabel.MISC_SOFT]`).
4. **No** mutation of module-level dicts at any point.

Acceptance:
- `overrides=None` path: all v0.7 tests remain byte-equal (104 default
  lane pass without the new test file added).
- `overrides=_RoomBuildOverrides(ceiling_label=MaterialLabel.CEILING_DRYWALL)`
  on Lecture_2 yields a `RoomModel` whose ceiling surface has
  `material == CEILING_DRYWALL` and whose floor / wall surfaces are
  unchanged.
- `_FURNITURE_BY_ROOM`, `_PIECE_EQUIVALENT_ABSORPTION_500HZ_M2`, and
  `_PIECE_EQUIVALENT_ABSORPTION_BANDS_M2` are byte-equal to v0.7
  (verified by `git diff` line count + a §2.5 sanity import test).
- ruff clean.

### §2.4 ADR 0015 — Lecture_2 bracketing methodology

**File**: `docs/adr/0015-lecture-2-ceiling-seat-bracketing.md` (NEW).

Required sections (consensus SHORT-mode protocol):
- Status (Accepted v0.8.0).
- Date (2026-05-09).
- Predecessor (v0.7.0 `b56ea7c`; ADR 0008..0014; D14..D18).
- Decision (V0..V3 [+ optional V4] sensitivity bracketing on
  Lecture_2; **NO** baseline change in v0.8; Scope A is measurement
  not commitment; bracketing perf doc lives at
  `docs/perf_verification_lecture2_bracket_2026-05-09.md`).
- Drivers (3-5 bullets):
  1. v0.6 / v0.7 perf doc shows Lecture_2 dominates 58–72 % of
     ex-BL corpus residual error per band; under-predicts every
     band (-0.908 s @500 Hz, -1.146 s @1 kHz, -1.054 s @2 kHz,
     -0.863 s @4 kHz).
  2. v0.7 critic verdict: 3 releases of "wait for non-canonical
     evidence" confuses unknown ground truth with unfalsifiable
     hypothesis; the corpus IS the truth-table.
  3. Bracketing yields a falsifiable answer: either a
     single-coefficient swap closes the gap (publishable positive)
     or it does not (publishable null, ruling out F3 + lecture_seat
     hypotheses and pointing to coupled-space / non-Sabine
     modelling).
- Alternatives considered (≥2):
  - (a) Bracketing as a library module under `roomestim/sensitivity/`
    — rejected (sensitivity work goes in `tests/` + `docs/`; library
    is for committed model behaviour; ADR 0014 alt-considered (b)
    precedent).
  - (b) F4a coefficient sweep at 2k/4k as the v0.8 headline — rejected
    (broader sweep without sharpened prior; deferred to v0.9 gated on
    Scope A outcome).
  - (c) Ratify the winning variant as v0.8 default — rejected
    (re-anchors on F3 entanglement; would change v0.6/v0.7 baseline
    without independent evidence; ratification reverse-trigger
    deferred to v0.9+).
  - (d) Ship Scope A only without the per-band MAE snapshot —
    rejected (snapshot is the infrastructure that makes future
    bracketing PRs auto-evaluated on residuals; critic M2 finding
    requires it).
  - (e) Write a coupled-space predictor (Cremer / Müller) — rejected
    (ADR 0014 alt-considered (b); needs per-sub-volume geometry; out
    of v0.8 scope).
- Why chosen — minimum-leverage answer to critic verdict; preserves
  v0.6/v0.7 numerical baseline; produces a publishable result whether
  positive or null; couples cleanly with snapshot infrastructure.
- Consequences:
  - `_build_room_model` gains an additive `overrides=` kwarg
    (default-equivalent path byte-equal to v0.7).
  - New default-lane perf doc at
    `docs/perf_verification_lecture2_bracket_2026-05-09.md`.
  - Future PRs editing predictor / adapter / per-band tables tripped
    by the per-band MAE snapshot; intentional updates require a
    one-line golden change + justification.
- Reverse-trigger / ratification reverse-if (v0.9+ may ratify a
  variant as new default iff ALL hold):
  1. Variant closes Lecture_2 |err\| ≤ 0.5 s @500 Hz.
  2. Variant does not regress {Lecture_1, Meeting_1, Meeting_2,
     Office_1, Office_2} by more than +0.10 s @500 Hz.
  3. Independent evidence (lab visit, author email, textbook citation
     beyond Beranek 2004 / Vorländer 2020) confirms the variant
     coefficient.
- References (Eaton 2016 TASLP §II-C; Vorländer 2020 §4.4 + §11;
  Beranek 2004 *Concert Halls and Opera Houses* Ch.3 Table 3.1; ADR
  0008, 0010, 0011, 0012, 0013, 0014; D14, D15, D16, D17, D18; v0.6
  perf doc `docs/perf_verification_e2e_2026-05-08.md`; project
  memory `project_taslp_2016_content.md`).

ADR 0014 References gains one cross-link line forward to ADR 0015
(parallel to the v0.7 cross-link pattern on ADR 0012 / 0013); other
ADRs untouched.
