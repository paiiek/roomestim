## Scope A — Lecture_2 ceiling/seat bracketing — DONE at v0.8.0

### Numerical result — null

| Variant | Lecture_2 Sabine 500 Hz pred (s) | Err vs measured 1.343 s | \|err\| Δ vs V0 |
| --- | ---: | ---: | ---: |
| V0 baseline | 0.435 | −0.908 | — |
| V1 ceiling=`ceiling_drywall` | 0.743 | −0.600 | **−0.308** (best single-coefficient) |
| V2 lecture_seat α₅₀₀=0.20 unoccupied | 0.322 | −1.021 | +0.113 (worse) |
| V3 V1+V2 combined | 0.464 | −0.879 | −0.029 |

Acceptance envelope was \|err\| ≤ 0.5 s @500 Hz Sabine on Lecture_2 with
≤ +0.10 s regression on each of {Lecture_1, Meeting_1, Meeting_2,
Office_1, Office_2}.

V3 fails BOTH acceptance clauses:
- \|−0.879\| s = 0.879 s > 0.5 s envelope.
- Meeting_1 +0.108 s and Meeting_2 +0.142 s @500 Hz Sabine, both >
  +0.10 s regression cap.

V1 (single-coefficient ceiling swap, the strongest individual move)
reduces \|err\| by 0.308 s but lands at −0.600 s — still outside the
envelope. This is the publishable v0.8 finding:
**single-coefficient ceiling/seat swap cannot close Lecture_2 without
external regression.**

### What this rules out and what it elevates

- **Rules out**: F3 ceiling-material reassignment as a *sufficient*
  explanation; `lecture_seat` α₅₀₀ revision as helpful (it goes the
  wrong way); the v0.7 audit's implied hope that "any single
  coefficient swap" would close Lecture_2.
- **Elevates** (now load-bearing for v0.9+):
  - F4a constrained per-band sensitivity sweep
    (`MaterialAbsorptionBands[wall_painted]`, `[ceiling_drywall]`,
    `[carpet]`) at 2k/4k.
  - Coupled-space predictor (Cremer / Müller two-room formula) for
    Lecture_2 — it exhibits the same multi-slope-decay signature as
    Building_Lobby, just less extreme.
  - Per-room Sabine-violation diagnostic (room-shape-dependent
    correction factor; ADR 0009 carries Eyring as parallel; v0.9+
    may add a *third* parallel for non-diffuse correction).

The v0.7 critic flagged that test count growth (80 → 84 → 100 → 104)

than via manual perf-doc diffing.

## ADR 0015 — Lecture_2 bracketing methodology — DONE at v0.8.0

`docs/adr/0015-lecture-2-ceiling-seat-bracketing.md` (NEW; +155 lines):
- **Status**: Accepted v0.8.0.
- **Decision**: V0..V3 (+ optional env-gated V4) sensitivity bracketing;
  NO baseline change in v0.8; bracketing is *measurement*, not
  *commitment*.
- **Drivers**: Lecture_2 dominates 58–72 % of ex-BL corpus residual
  error per band; v0.7 critic verdict; bracketing yields a falsifiable
  answer regardless of outcome.
- **Alternatives considered**: 5 (a..e) — library-module residence
  rejected; F4a sweep as v0.8 headline rejected (deferred); ratification
  rejected (would re-anchor on F3 entanglement); Scope A only without
  Scope B rejected (snapshot is the residual-coverage infra); coupled-
  space predictor rejected (out of v0.8 scope).
- **Consequences**: `_build_room_model` gains additive `overrides=`
  kwarg; new appendix perf doc; future PRs tripped by snapshot test.
- **Reverse-trigger** (v0.9+ may ratify a winning variant iff ALL):
  (i) variant closes Lecture_2 \|err\| ≤ 0.5 s @500 Hz; (ii) variant
  does not regress {Lecture_1, Meeting_1, Meeting_2, Office_1,
  Office_2} by > +0.10 s @500 Hz; (iii) independent evidence (lab
  visit, author email, textbook beyond Beranek 2004 / Vorländer 2020)
  confirms the variant coefficient. **None of (i)..(iii) is satisfied
  at v0.8 ship; ratification stays gated.**

ADR 0014 References gains exactly one cross-link line forward to ADR
0015 (parallel to the v0.7 ADR 0012 / 0013 → ADR 0014 pattern).
