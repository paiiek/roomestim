장애물-인지 배치 (참고)
⚠ 결정론적 기하 휴리스틱 — 음향·최적성·청감 검증이 아닙니다.
  • 여유거리: 가구를 '바닥~높이' 직육면체(회전 무시)로 보고 3D 거리로 계산 — 천장·벽 상단이 낮은 가구 위면 통과(평면 겹침으로 잘못 거부하지 않음).
  • 시야(LOS)도 높이 인지: 스피커·귀가 낮은 가구보다 위면 가림으로 보지 않음(회절 무시, 각 물체는 불투명 상자).
  • Mode A (format_avoid): 표준 각도 유지 + 최소 각도만 틀어 회피 — 채널별 편차 표시, 못 풀면 UNRESOLVED로 두고 임의로 옮기지 않음.
  • Mode B (coverage_avoid): 벽/천장 후보에서 장애물·가림을 3D 제거 후 DBAP 커버리지 최대화 — SPL 주장 없음. 후보 부족 시 k/N만 배치+고지, 0개일 때만 실패.
  • 각도 근거: 공개 자료(ITU-R BS.775, Dolby Atmos Home) — 유료 표준 아니며 방에서 추론하지 않음.
